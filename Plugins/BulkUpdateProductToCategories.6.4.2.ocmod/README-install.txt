Thank you for purchasing our extension!
Installation instructions:

1.Login to your admin panel and navigate to: Extensions >  Extension Installer
2.Click Upload and select the "BulkUpdateProductToCategories_ver6.2.ocmod.zip" file and wait until finish.
3.Navigate to: Extensions > Modules > (install) "Bulk Update Product To Categories". Edit and configure the module. Please make sure that status is set to: Enabled.
4.Naviage to Extensions > Modifications and click the Refresh button.

This is It! Now you have your "Bulk Update Product To Categories" installed!! Congratulations and enjoy!
